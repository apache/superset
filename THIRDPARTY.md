# THIRDPARTY.md - Inventário de Soluções de Terceiros

Este arquivo lista as principais dependências e soluções de terceiros utilizadas no projeto Loonar, garantindo transparência e rastreabilidade de licenciamento e segurança.

| Solução | Licença | Versão | Link | Descrição |
| :--- | :--- | :--- | :--- | :--- |

| **Nginx (Alpine)** | BSD 2-Clause | 1.28.2 | [Nginx](https://nginx.org/) | Proxy reverso e servidor web leve. |
| **PostgreSQL** | PostgreSQL | ~16 | [PostgreSQL](https://www.postgresql.org/) | Banco de dados relacional (metadados). |

| **Microsoft ODBC Driver** | MS EULA | 17 / 18 | [Microsoft Docs](https://learn.microsoft.com/en-us/sql/connect/odbc/download-odbc-driver-for-sql-server) | Driver para conexão com SQL Server. |

| **Alpine Linux** | MIT/GPL/Others | v3.20 | [Alpine Linux](https://alpinelinux.org/) | Imagem base otimizada para containers. |
