# Alert

Alert component for displaying important messages to users. Wraps Ant Design Alert with sensible defaults and improved accessibility.

## Installation

This component is available in the `@apache-superset/core` package, which is automatically available to Superset extensions.

## Usage

```tsx
import { Alert } from '@apache-superset/core';

function MyExtension() {
  return (
    <Alert>
      {/* Your content here */}
    </Alert>
  );
}
```
