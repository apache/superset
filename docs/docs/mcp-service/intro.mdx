# Superset MCP Service

The Superset Model Context Protocol (MCP) service provides programmatic access to Superset dashboards, charts, datasets, and instance metadata. Built for LLM agents and automation tools.

## What is MCP?

The Model Context Protocol (MCP) is an open standard that allows AI assistants to securely connect to data sources and tools. Superset's MCP service exposes **20 production-ready tools**, **2 guided prompts**, and **2 contextual resources** that enable:

- ğŸ“Š **Data Exploration**: List and query dashboards, charts, and datasets

## Quick Start

### Installation

:::note
The MCP service is included with Superset development setup. FastMCP dependencies are installed with `pip install -e .[development,fastmcp]`.
:::

```bash
# Using Docker Compose (recommended)
docker-compose -f docker-compose-light.yml --profile mcp build
docker-compose -f docker-compose-light.yml --profile mcp up -d

# Or manually with development setup
git clone https://github.com/apache/superset.git
cd superset
python3 -m venv venv
source venv/bin/activate
pip install -e .[development,fastmcp]

# Initialize database
superset db upgrade
superset init

# Start Superset
superset run -p 8088 --with-threads --reload --debugger

# Start MCP service (separate terminal)
source venv/bin/activate
superset mcp run --port 5008 --debug
```

### Connecting to MCP

#### Option 1: ModelContextChat.com (No Installation Required)

The fastest way to start using Superset MCP is through [ModelContextChat.com](https://modelcontextchat.com):

1. Navigate to ModelContextChat.com
2. Configure MCP connection to `http://localhost:5008`
3. Start chatting with Claude about your Superset data

See the [MCP Service README](https://github.com/apache/superset/tree/master/superset/mcp_service#-using-modelcontextchatcom-web-based-mcp-client) for detailed instructions with screenshots.

#### Option 2: Claude Desktop Integration

```json

}
```

## Example Usage

### Tools
```python
# List dashboards
dashboards = client.call_tool("list_dashboards", {
    "search": "sales",
    "page_size": 10
})

# Create a chart
chart = client.call_tool("generate_chart", {
    "dataset_id": "1",
    "config": {
        "chart_type": "line",
        "x": {"name": "date"},
        "y": [{"name": "revenue", "aggregate": "SUM"}]
    }
})

# Export chart data
data = client.call_tool("get_chart_data", {
    "identifier": chart["chart_id"],
    "format": "json",
    "limit": 1000
})

# Execute SQL query
result = client.call_tool("execute_sql", {
    "database_id": 1,
    "sql": "SELECT * FROM sales WHERE year = {year}",
    "parameters": {"year": "2024"},
    "limit": 100
})
```

### Prompts
```python
# Get personalized onboarding
prompt = client.get_prompt("superset_quickstart", {
    "user_type": "analyst",
    "focus_area": "sales"
})

# Get chart creation guidance
prompt = client.get_prompt("create_chart_guided", {
    "chart_type": "bar",
    "business_goal": "reporting"
})
```

### Resources
```python
# Get instance metadata
metadata = client.read_resource("superset://instance/metadata")
# Returns instance stats, popular datasets, recent dashboards

# Get chart templates
templates = client.read_resource("superset://chart/templates")
# Returns pre-configured templates, color schemes, best practices
```
